## 📌 Project Overview
This project is the foundational task of my AI/ML internship, focusing on **Exploratory Data Analysis (EDA)**. The goal was to perform a deep dive into the classic **Iris Dataset**, identifying patterns, distributions, and relationships between floral features like Sepal and Petal dimensions across different species.

---

## 🛠️ Technical Workflow & Implementation

### 1. Data Acquisition & Inspection
* **Source**: Loaded the `Iris` dataset directly using the Seaborn library.
* **Dimensions**: Analyzed 150 rows and 5 columns (Sepal Length, Sepal Width, Petal Length, Petal Width, and Species).
* **Statistical Summary**: Performed `df.describe()` to understand mean, standard deviation, and quartiles of the dataset.

### 2. Data Visualization Strategy
We utilized multiple plotting techniques to extract insights:
* **Scatter Plots**: Used to visualize the relationship between Sepal Length and Width, with hue coding to distinguish between *Setosa*, *Versicolor*, and *Virginica*.
* **Histograms**: Created distribution plots for all numerical features to understand the frequency and spread of data points.
* **Box Plots**: Implemented to detect outliers and compare the distribution of Petal Length across different species.

---

## 📊 Key Insights & Findings

* **Species Separation**: Setosa species is easily distinguishable from others based on petal dimensions.
* **Feature Correlation**: Petal length and petal width show a strong positive correlation across all species.
* **Data Quality**: The dataset was found to be clean with no missing values, making it ideal for visualization and future modeling.
